# Contributing to GaiaComCoin

Thank you for your interest in contributing to GaiaComCoin.

This project values transparency, security, and responsible collaboration.

---

## 1. General Rules

By contributing, you agree that:

* Contributions are voluntary
* No compensation is guaranteed
* You have the right to submit the contributed code

---

## 2. Contribution Types

We welcome:

* Bug fixes
* Security improvements
* Performance optimizations
* Documentation improvements

All contributions must:

* Follow Rust best practices
* Avoid unsafe code unless explicitly justified
* Pass CI and security checks

---

## 3. Security Contributions

If you discover a security vulnerability:

* Do NOT open a public issue
* Contact the maintainers privately

Responsible disclosure is required.

---

## 4. Code Review

All contributions are subject to:

* Technical review
* Security review
* Governance compatibility review

Maintainers reserve the right to reject contributions that conflict with protocol principles.

---

## 5. Licensing

By submitting a contribution, you agree that your contribution:

* Will be licensed under the license applicable to the target directory
* May be redistributed as part of the project

See LICENSE.md for details.

---

## 6. No Liability

Contributors are not responsible for downstream use of their contributions.

---

End of Contribution Guidelines.
